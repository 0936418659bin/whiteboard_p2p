## Technical Specifications

### Architecture
- **Pattern**: Modular, layered architecture
- **Threading**: Async network I/O with dedicated threads
- **Communication**: Message-based via TCP sockets
- **Discovery**: UDP broadcast for peer detection
- **Serialization**: Java object serialization

### Technology Stack
| Component | Technology | Version |
|-----------|-----------|---------|
| Language | Java | 17+ |
| Build System | Maven | 3.6+ |
| UI Framework | Swing | Built-in |
| UI Theme | FlatLaf | 3.2.5 |
| Networking | Java NIO/Sockets | Built-in |

### Network Ports
- **Discovery**: UDP 55555 (broadcast), 55556 (listen)
- **P2P Connections**: TCP 55550-55559

### System Requirements
- **Java**: 17 or higher
- **RAM**: 512MB minimum, 2GB recommended
- **Disk**: 100MB for application + dependencies
- **Network**: Local LAN for peer discovery
- **OS**: Windows, Linux, macOS

## Troubleshooting at a Glance

| Issue | Solution |
|-------|----------|
| Java not found | Install Java 17+ from jdk.java.net |
| Maven not found | Install Maven from maven.apache.org |
| Port already in use | Close other instances or change port |
| Peers not discovered | Check firewall, ensure same network |
| Drawing not syncing | Verify connection shows "Connected: N" |
| Out of memory | Increase heap: `java -Xmx1g ...` |

**Full troubleshooting**: See QUICKSTART.md and INSTALLATION.md
